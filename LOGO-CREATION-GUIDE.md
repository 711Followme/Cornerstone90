# Converting Cornerstone: 90 Icon to PNG

## Current Icon
The app already has a vector SVG icon at:
- `public/cornerstone-icon.svg`

This is a simple cornerstone square shape in gold/blue that works well.

## Convert to PNG Online (FREE)

### Method 1: CloudConvert (Recommended)
1. Go to: https://cloudconvert.com/svg-to-png
2. Upload `public/cornerstone-icon.svg`
3. Set width to 1024px (or 512px, 192px as needed)
4. Click "Convert"
5. Download PNG file
6. Rename and place in `/public/` folder

### Method 2: Convertio
1. Go to: https://convertio.co/svg-png/
2. Upload the SVG
3. Choose size (1024x1024 for best quality)
4. Convert and download

### Method 3: Using Browser (Quick)
1. Open `public/cornerstone-icon.svg` in Chrome
2. Right-click → Inspect
3. Right-click on the SVG element
4. "Capture node screenshot"
5. Resize to 512x512 or 192x192

## Sizes Needed

For a complete PWA, create these PNG files:

- `cornerstone-icon-192.png` (192x192px) - Required for PWA
- `cornerstone-icon-512.png` (512x512px) - Required for PWA
- `cornerstone-icon-1024.png` (1024x1024px) - High quality master
- `favicon.png` (32x32px) - Browser tab icon
